P1217 [USACO1.5] 回文质数 Prime Palindromes

做这题时全是TEL，于是学习了下筛法与打表

筛法：三种素数筛法(普通筛法、埃氏筛法、欧拉筛法)

埃拉托斯特尼筛法（Sieve of Eratosthenes）是一种用于找出一定范围内所有素数的古老算法。它的基本思想是从小到大遍历数字，将素数的倍数标记为合数，从而筛选出素数。码首先创建了一个布尔数组 isPrime，用于标记每个数字是否为素数。然后，它从2开始遍历到 n 的平方根，对于每个素数 p，将其倍数标记为合数。最后，遍历布尔数组，找到所有标记为素数的数字，它们就是小于等于 n 的素数。
埃拉托斯特尼筛法的时间复杂度是 O(n log log n)，这使得它是一种非常高效的素数筛选算法。

回文数：回文数题解

要检查一个整数是否是回文数，可以使用一种更有效的方法而不是反转整个数字。以下是一种优化算法：

处理特殊情况：
如果输入是负数，它不可能是回文数，因为负数的反转后的数字不会等于原始数字。
如果输入以0结尾（除了0本身），也不可能是回文数，因为不会存在以0开头的数字。
通过比较数字的前半部分和后半部分来检查回文性：
将整数分成前半部分和后半部分，例如，对于整数12321，前半部分是123，后半部分是21。
反转后半部分数字，然后与前半部分数字进行比较。
如果它们相等，那么这个数字是回文数。
这种算法只需要反转一半的数字，从而减少了计算的时间和内存开销。

对于回文质数，它不可能是偶数位（除11）

具有偶数个数位的回文数一定是 11 的倍数

证明：具备偶数个数位回文数一定是11的倍数证明

具体代码：

#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdbool.h>
#include<stdlib.h>
bool weishu(int k)  //位数
{
	if (k >= 10 && k < 100 && k != 11 || k >= 1000 && k < 10000)return false;
	else if (k >= 100000 && k < 1000000 || k >= 10000000 && k < 100000000)return false;
	else return true;
}
bool isPalindrome(int x) {
	if ( (x % 10 == 0)) {
		return false; // 负数或以0结尾的数字不是回文数
	}

	int reversedHalf = 0;
	while (x > reversedHalf) {
		int digit = x % 10;
		reversedHalf = reversedHalf * 10 + digit;
		x /= 10;
	}

	// 如果x的位数是奇数，将reversedHalf除以10后与x比较
	return x == reversedHalf || x == reversedHalf / 10;
}
int main()
{
	FILE* file = fopen("E:\\123.txt", "w");
	int a, n;
	scanf("%d%d", &a, &n);
	bool* isPrime = (bool*)malloc(sizeof(int) * (n + 1));
	for (int i = 0; i <= n; i++) {
		isPrime[i] = true; // 假设每个数字都是素数
	}
	isPrime[0] = isPrime[1] = false; // 0和1不是素数

	for (int p = 2; p * p <= n; p++) {
		if (isPrime[p] == true) {
			// 如果p是素数，将p的倍数标记为合数
			for (int i = p * p; i <= n; i += p) {
				isPrime[i] = false;
			}
		}
	}
	if (a % 2 == 0) a++;
	for (int i = a; i <= n; i+=2)
	{
		if (i == 9989900)
		{//如果到了这个数，就break 
			break;
		}
		if (weishu(i)&&isPalindrome(i) && isPrime[i])
		{
			printf("%d\n", i); 
			fprintf(file, "%d,",i);
		}
	}
	fclose(file);
	return 0;
}
