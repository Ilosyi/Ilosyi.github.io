一、选择排序

选择排序（Selection Sort）是一种简单直观的排序算法，其基本思想是在未排序的部分中选择最小（或最大）的元素，然后将其放到已排序部分的末尾。这个过程不断地重复，直到所有元素都排序完成。
选择排序的动画演示
/代码实现/

   void selectionSort(int arr[], int n) {
   int i, j, minIndex, temp;

   // 遍历数组
   for (i = 0; i < n - 1; i++) {
   // 假设当前索引为最小值索引
   minIndex = i;

   // 在未排序部分中寻找最小值的索引
           for (j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
               minIndex = j;
           }
       }

   // 将找到的最小值与当前位置交换
       temp = arr[minIndex];
       arr[minIndex] = arr[i];
       arr[i] = temp;
    }
   }

二、冒泡排序

普通冒泡排序

冒泡排序是一种简单的比较排序算法，它反复遍历待排序的元素列表，比较相邻的元素，并将它们交换，直到整个列表按照升序或降序排列。它得名于比较过程中较小（或较大）的元素像气泡一样"浮"到数组的顶端。冒泡排序的时间复杂度为O(n^2)，其中n是元素的数量，因此它在大数据集上的性能不佳，但对于小数据集来说是一种简单的排序方法。
交换原则：两两比较进行交换，如果小的在后，大的在前即交换
冒泡排序的动画演示
/代码实现/

   #include <stdio.h>
   void bubbleSort(int arr[], int n) {
    int temp;
   for (int i = 0; i < n - 1; i++) {
   for (int j = 0; j < n - i - 1; j++) {
   if (arr[j] > arr[j + 1]) {
   // 交换arr[j]和arr[j+1]
   temp = arr[j];
   arr[j] = arr[j + 1];
   arr[j + 1] = temp;
   }
    }
    }
    }

改进冒泡排序

冒泡排序改进1：在某次遍历中如果没有数据交换，说明整个数组已经有序。因此通过设置标志位来记录此次遍历有无数据交换就可以判断是否要继续循环。
冒泡排序改进2：记录某次遍历时最后发生数据交换的位置，这个位置之后的数据显然已经有序了。因此通过记录最后发生数据交换的位置就可以确定下次循环的范围了。
/代码实现/

   #include <stdio.h>
   void bubbleSort(int arr[], int n) {
   int temp;
   int lastSwap = n - 1; // 初始化最后一次交换位置为数组的最后一个元素
   for (int i = 0; i < n - 1; i++) {
   int newLastSwap = -1; // 初始化新的最后一次交换位置为-1，用于判断是否有交换发生
   for (int j = 0; j < lastSwap; j++) {
   if (arr[j] > arr[j + 1]) {
    // 交换arr[j]和arr[j+1]
   temp = arr[j];
   arr[j] = arr[j + 1];
   arr[j + 1] = temp;
   newLastSwap = j; // 更新最后一次交换的位置
   }
   }
   // 如果没有数据交换，说明整个数组已经有序，可以提前退出排序
   if (newLastSwap == -1) {
   break;
    }
   lastSwap = newLastSwap; // 更新最后一次交换的位置
    }
   }

三、快速排序

推荐一个讲解视频：全网最清晰快速排序，看完快排思想和代码全部通透，不通透你打我！
快速排序的基本思想是选择一个基准元素，将数组分成两部分，左边的部分包含所有小于基准的元素，右边的部分包含所有大于基准的元素，然后对这两部分分别进行快速排序，最终将整个数组排序。
快速排序的步骤如下：
1.选择基准元素： 从数组中选择一个基准元素（通常是第一个元素、最后一个元素或者随机选择），记作pivot。
2.分区（Partition）： 将数组分成两部分，小于基准元素的放在左边，大于基准元素的放在右边，相等的可以放在任意一边。分区过程可以使用双指针法来实现。
3.递归排序： 递归地对左右两部分进行快速排序。
4.合并： 由于每次递归调用都会将一个元素放置到其最终位置，所以当递归调用完成时，整个数组就被排序好了。
这个过程是递归的，当数组被分割为单个元素时，递归结束。
快速排序的优势在于它的平均时间复杂度为O(n log n)，而且它是原地排序（in-place sorting），只需要常数级别的额外空间。但是，快速排序的性能高度依赖于选择的基准元素，如果选择的基准元素不合适，可能会导致算法的性能下降。为了避免这种情况，通常使用随机选择基准元素或者使用三数取中法（选择第一个、中间一个和最后一个元素中的中间值作为基准）来提高算法的稳定性。
